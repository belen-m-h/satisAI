# Installation and run

    The code was developed with Python 3.8.2
    In order to run it:
    - Use the virtual environment "satis_env" on the folder or create a new one with Python 3.8.2 and install the dependecies:
            pip install -r requirements.txt

    - Run the code with:
            python main.py


# Testing
    Execute the tests with:
            python -m pytest test/unit_tests.py
